# 🍜 ManaBisai

> **"Mana Bisai?"** — Brunei Malay for *"Where's the good food?"*

ManaBisai is a GPS-powered food discovery platform built for Brunei. It helps locals and visitors find nearby food stalls, restaurants, and hidden gems — all in one place, tailored for the Bruneian context.

---

## 📱 What It Does

- 📍 **Discover food spots** near you using real-time GPS
- 🗺️ Browse an interactive map of Brunei's food scene
- 🔍 Search and filter by cuisine, location, or rating
- 🛠️ Admin panel to manage listings, users, and data

---

## 🧱 Tech Stack (MEVEN)

| Layer | Technology |
|---|---|
| Mobile Client | Expo (React Native) |
| Admin Panel | Vue.js 2 + Vuetify |
| Backend API | Node.js + Express.js |
| Database | MySQL (primary) |
| Auth | JWT (rotate on refresh) |

> **MEVEN** = **M**ySQL · **E**xpress · **V**ue · **E**xpo · **N**ode
